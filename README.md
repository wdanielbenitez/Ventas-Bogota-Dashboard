# Ventas-Bogota-Dashboard 2023-2025 (Interactive Dashboard using MS Excel)

## Objetivo del proyecto
Analizar patrones de ventas de productos tecnológicos en Bogotá, identificar canales más rentables y localidades con mayor demanda.
Esto basado en información de los años 2023-2025.

## Dataset usado
- <a href="https://github.com/wdanielbenitez/Ventas-Bogota-Dashboard/blob/master/DatasetVentasBogota.xlsx">Dataset</a>

## KPIs Principales
- **Ventas Totales**: $12,605,132 COP
- **Transacciones**: 400 ventas
- **Ticket Promedio**: $31,512 COP
- **Productos Vendidos**: 10 activos

## Insights Clave
1. **Celular lidera ventas** con 56 unidades vendidas
2. **Tienda Online supera a Física** en volumen de ventas
3. **2023 fue el mejor año** con $4.5M en ingresos
4. **Teusaquillo y Chapinero** son las localidades top

## Herramientas
- Microsoft Excel
- Tablas Dinámicas
- Gráficos Interactivos
- Slicers para filtrado dinámico

## Estructura
- `Data` - Datos fuente
- `Tablas Dinamicas` - Análisis en tablas Dinámicas y fórmulas
- `Dashboard` - Vista ejecutiva principal
  
## Proceso
- Verificar los datos, que el archivo no este corrupto, datos en blanco, anomalías.
- Asegurar que los datos sean consistentes y limpios de acuerdo a sus tipos de dato, formato.
- Crear tablas dinámicas de acuerdo al objetivo del proyecto y preguntas propuestas.
- Generar un dashboard donde se visualice de manera eficiente el resultado de transformar los datos.
  
## Cómo Usar
1. Abrir archivo Excel (VentasBogota.xlsx)
2. Usar slicers (Año, Localidad, Canal) para filtrar la información
3. Los gráficos se actualizan automáticamente
